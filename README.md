# Seestar_nina

Seestar Nina is a plugin for N.I.N.A that allows imaging on the Seestar without transferring the images to the source computer. This speeds up aquisition time.
The plugin also allow for the use of the SesStar built-in Autofocus routine instead of the NINA routine.
